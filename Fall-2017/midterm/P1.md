## سوال اول
تعریف تابعی به صورت زیر را بنویسید:

```c
int check (float side1, float side2, float angle)
```

که با دریافت دو ضلع و زاویه‌ی بین آن‌ها (به درجه) از کاربر خروجی طبق جدول زیر بر می‌گرداند:

| شکل                         | خروجی |
| --------------------------- | ----- |
| مربع                        | 1     |
| لوزی                        | 2     |
| مستطیل                      | 3     |
| نه مربع، نه لوزی، نه مستطیل | 4     |
