## سوال اول
با فرض اينكه مقادير p، q، r و s به صورت زير باشد، مقدار z را در هر يك از موارد ذيل مشخص كنيد.

```c
bool p = false, q;
int r, s;
q = false; r = 0; s = 100;

(a) z = p && q || q;

(b) z = !p && !q + r;

(c) z = (p + r + s) && q;

(d) z = p >= q => r && s;

(e) z = !r % s %!p % !q;

(f) z = p ? q ? r : s : 2;
```
