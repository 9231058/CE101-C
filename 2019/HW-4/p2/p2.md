# سوال دوم (۱۵ امتیاز)

## مساله

برنامه‌ای بنویسید که اعداد n و m را از کاربر بگیرد و اعداد ۱ تا n را m بار به صورت تصادفی چاپ کند.
برای مثال اگر 5 = n و m = 3 باشد خروجی شبیه به این تولید خواهد شد:

```sh
1 5 3 2 4
2 3 1 5 4
5 3 4 1 2
```

## ساختار ورودی

```sh
<n> <m>
```

## ساختار خروجی

```sh
<x> ... <x>
...
<x> ... <x>
```
