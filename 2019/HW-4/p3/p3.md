```c
x(n) = (n > 1) ? n^n: 0
x(n) = (n > 1) ? 2*n + x(n - 1) : 0
x(n) = (n > 1) ? n * (n + 1) : 1
```

```c

x(2) = 2 * 2 + x(1) = 2 * 2 + 0 = 4

```
