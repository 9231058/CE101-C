# سوال پنجم (۲۰ امتیاز)

## مساله

برنامه‌ای بنویسيد شش عدد a1 و b1 و a2 و b2 و a3 و b3 را به عنوان ورودی بگيرد و ميزان همپوشانی سه بازه‌ی  [b1, a1] و [b2, a2] و [b3, a3] را به عنوان خروجی چاپ كند.

## ساختار ورودی

```sh
<a1> <b1>
<a2> <b2>
<a3> <b3>
```

## ساختار خروجی

```sh
<X>
```

## نمونه ورودی ۱

```sh
0 2
1 4
0 5
```


## نمونه خروجی ۱

```sh
1
```
