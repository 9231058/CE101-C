# سوال اول (۱۵ امتیاز)

## مساله

با فرض اينكه مقادير p، q، r و s به صورت زير باشد، مقدار z را در هر یک از موارد ذيل مشخص كنيد.

```c
bool p, q;

int r, s;

p = false; q = (0 == 0); r = 0; s = 20;
```

الف)

```c
z = (q && p) || s + 1
```

ب)

```c
z = !(s && q <= p)
```

ج)

```c
z = !p + !s
```

د)

```c
z = !p >= !q + !p + 1 && !q
```

هـ)

```c
z = !s && !(!p && ++r)
```

و)

```c
z = (p != q - 1) % (q <= !r)
```
