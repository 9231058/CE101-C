# سوال دوم (۱۵ امتیاز)

## مساله

نحوه توليد خروجی هر قطعه كد را مرحله به مرحله شرح دهيد و مواردی كه cast رخ می‌دهد را مشخص كنيد.

قسمت الف)

```c
int i, j; double d; float f;
i = d = j = f = 10;
i = d * j + 1.0;
j = (int) i % (int)((double)f * j);
printf("%d, %d\n", i, j);
```
