# سوال سوم (۲۰ امتیاز)

## مساله

برنامه‌ای بنويسيد كه چهار عدد را از كاربر بگيرد و آنها را با كمترين تعداد مقايسه به صورت نزولی مرتب كرده و چاپ كند.

## ساختار ورودی

```sh
<x1> <x2> <x3> <x4>
```

## ساختار خروجی

```sh
<x1> <x2> <x3> <x4>
```

## نمونه ورودی ۱

```sh
1 2 3 4
```


## نمونه خروجی ۱

```sh
4 3 2 1
```
