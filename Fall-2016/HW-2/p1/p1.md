
مقدار و نوع خروجی عبارت‌های زیر را مشخص کنید:

```c
float a = 10, b = 20;
int c = 30, d = 40, e = 2, f = 5, g = 10;
```

الف)

```c
(((a) * (b)) / ((c + d)))
```

ب)

```c
((a * b) - (c % d))
```

ج)

```c
((a – (d % c)) / ((d) * (e + f)))
```

د)

```c
(a * b) - (c / d) - (e / (f / g))
```

هـ)

```c
((((a + b) + c) * d) + ((e + f) – g)) * a
```
