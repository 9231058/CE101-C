```c
bool p = true, q;
int r, s;
q = (1 != 1);
r = 0;
s = 100;
```

```c
z = (q || !p) && s + r
```

(b)

```c
z = !(r <= q < s) ? p : !p
```

(c)

```c

```

(d)

```c
z = !p <= !q + p || !p + q
```

(e)

```c
z = !s && p && !(!p && r)
```

```c

```
