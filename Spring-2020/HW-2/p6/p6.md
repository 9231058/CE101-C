ترتیب
ورودی‌ها به صورت زیر است. ورودی‌های m
و c از نوع float است.

a b c r m q

$$
Z  =   (8.8(a  +  b)2  /  c  -  0.5  +  2a  /  (q  +  r)) / ((a + b)   * (1  /  m))
$$

$$
X =   ( - b  +  (b  *  b)   +  2.4ac) / 2a
