# سوال سوم
## مساله
عدد صحیح n به شما داده می‌شود. برنامه‌ای بنویسید که مشخص کند n توانی از ۲ می‌باشد یا خیر؟
## ساختار ورودی

```sh
<n>
```

## ساختار خروجی

```sh
true/false
```

## ورودی نمونه ۱

```sh
16
```

## خروجی نمونه ۱

```sh
true
```

## ورودی نمونه ۲

```sh
218
```

## خروجی نمونه ۲

```sh
false
```
