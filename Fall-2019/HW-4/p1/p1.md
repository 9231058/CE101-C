# سوال اول (امتیازی)
## مساله 
تعداد n  نقطه روی صفحه‌ی دو بعدی داده شده است. برنامه‌ای بنویسید که بیشترین تعداد نقاطی که روی یک خط راست قرار می‌گیرند را گزارش کند.

## ورودی نمونه ۱

```sh
3
1 1
2 2
3 3
```

## خروجی نمونه ۱

```sh
3
```

در مثال فوق:

```
^
|
|        o
|     o
|  o  
+------------->
0  1  2  3  4
```
