# سوال چهارم
## مساله

برنامه‌ای بنویسید که با دریافت عدد صحیح و نامنفی c مشخص کند می‌تواند این عدد را به صورت زیر نوشت:

$$
a^2+b^2=c
$$

در رابطه فوق a و b اعداد صحیح می‌باشند.

## ساختار ورودی

```sh
<c>
```

## ساختار خروجی

```sh
true/false
```

## ورودی نمونه ۱

```sh
5
```

## خروجی نمونه ۱

```sh
true
```
