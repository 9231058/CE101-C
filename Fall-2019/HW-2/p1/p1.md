## مساله

مقدار و نوع خروجی نهایی عبارتهای زیر را مشخص کنید:

```c
float a = 10.2, b = 20.1;
short c = 200, d = 40, e = 20, f = 15, g = 110;
```


الف)

```c
((a * b) / c) + ((e / f) + 100)
```


ب)

```c
((c % f) - (b / a) - g)
```


ج)

```c
(a + b + c / d / e / f + g)
```


د)
```c
(a + g) / (b * f) + (a - (f / g))
```


هـ)
```c
((((g - c) % f) * d) + ((a + b) – a)) * c
```
