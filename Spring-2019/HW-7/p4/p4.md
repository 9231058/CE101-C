# سوال چهارم

## مساله

برنامه‌ای بنویسید که یک تعداد عدد گویا که نمیدانیم تعداد آنها چند است را از کاربر بگیرد. میدانیم که این لیست از اعداد گویا با عدد 0 / 0 تمام میشوند. اعدادی را که کمتر از x برابر جمع کل اعداد هستند را چاپ کند. x به عنوان ورودی از کاربر گرفته میشود.

## ساختار ورودی

```sh
<a1>/<b1>
<a2>/<b2>
…
0/0
<x>
```

## ساختار خروجی

```sh
<x1>/<y1>
…
<xn>/<yn>
```

## ورودی نمونه ۱

```sh
1 1
1 1
0 0
1
```

## خروجی نمونه ۱

```sh
1 / 1
1 / 1
```
