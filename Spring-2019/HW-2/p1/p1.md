## مساله

مقدار و نوع خروجی نهایی عبارت‌های زیر را مشخص کنید:

```c
int a = 1, b = 2;
double c = 300.1, d = 4, e = 500.5, f = 600, g = 7.7; 
```


الف)

```c
((a * b) / c) + ((e / f) + 100)
```


ب)

```c
((c % f) - (b / a) - g)
```


ج)

```c
(a + b + c / d / e / f + g)
```


د)
```c
(a + g) / (b * f) + (a - (f / g))
```


هـ)
```c
((((g - c) % f) * d) + ((a + b) – a)) * c
```
