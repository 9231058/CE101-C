# سوال پنجم (۲۰ امتیاز)

## مساله

برنامه‌ای بنویسید چهار عدد a1 و b1 و a2 و b2 را به عنوان ورودی بگیرد و میزان همپوشانی دو بازه [a1, b1] و            [a2, b2] را به عنوان خروچی چاپ کند.

## ساختار ورودی

```sh
<a1> <b1>
<a2> <b2>
```


## ساختار خروجی

```sh
<intersection>
```

## نمونه ورودی ۱

```sh
0 0
1 1
```


## نمونه خروجی ۱

```sh
0
```

## نمونه ورودی ۲

```sh
0 3
1 2
```


## نمونه خروجی ۲

```sh
1
```


## نمونه ورودی ۳

```sh
-10 10
-7 3
```

## نمونه خروجی ۳

```sh
10
```
